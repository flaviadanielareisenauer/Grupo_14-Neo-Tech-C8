## Temática de la web y audiencia objetivo

El sitio será una plataforma de venta de productos de informática y tecnología (computadoras, tablets, celulares, accesorios gamers, etc), en la cual los clientes también podrá contar con armado y arreglo de pc. 

La audiencia objetivo serán personas de todas las edades.
